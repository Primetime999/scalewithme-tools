# ScaleWithMe Tools

Free, invokable tools for builders — the commercial layer that turns what you build into
something you can sell. Install once inside Claude Code, then just invoke them.

## Install

In Claude Code:

```
/plugin marketplace add Primetime999/scalewithme-tools
/plugin install avatar-builder@scalewithme
```

Then invoke it by asking for what it does, e.g. *"help me build my customer avatar"* — or run it
directly.

## Tools

| Tool | What it does |
|------|--------------|
| **avatar-builder** | Interviews you about your product and outputs a sharp best-fit-customer thesis — best-fit customer, acute problem, current alternative, their exact language, anti-fit, a one-line thesis, and the riskiest assumption to validate first. |

More tools ship here over time (naming + domain, positioning, messaging, pricing).
